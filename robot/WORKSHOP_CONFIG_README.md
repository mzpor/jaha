# 📚 راهنمای کانفیگ کارگاه‌ها

## 🔧 تغییرات اعمال شده

### 1. تغییر نام ظاهری کارگاه‌ها
- **قبل:** `WORKSHOP_DISPLAY_NAME: 'راهبران'`
- **بعد:** `WORKSHOP_DISPLAY_NAME: 'راهبران'` ✅

### 2. تغییر نام‌های فیلدها
- **هزینه** → **منطقه** (موقتاً غیرفعال)
- **لینک** → **منطقه**

### 3. غیرفعال کردن فیلد هزینه (منطقه)
- **قبل:** `COST: 1` (اجباری)
- **بعد:** `COST: 0` (اختیاری - موقتاً غیرفعال)

### 4. کانفیگ نام‌های قابل تغییر
```javascript
DISPLAY_NAMES: {
  WORKSHOP: 'راهبران',         // نام کارگاه - قابل تغییر
  INSTRUCTOR: 'راهبر',         // نام مربی - قابل تغییر
  ASSISTANT: 'کمک مربی',       // نام کمک مربی - قابل تغییر
  STUDENT: 'فعال'              // نام دانش‌آموز - قابل تغییر
}
```

## 🎯 هدف تغییرات

1. **هماهنگی با کانفیگ نقش‌ها:** نام کارگاه‌ها با `COACH: 'راهبر'` هماهنگ است
2. **تغییر نام فیلدها:** "هزینه" و "لینک" به "منطقه" تغییر یافته
3. **غیرفعال کردن موقت:** فیلد هزینه (منطقه) موقتاً غیرفعال شده
4. **قابلیت تغییر آسان:** می‌توانید نام‌ها را بدون تغییر کد تغییر دهید

## 📝 نحوه تغییر نام‌ها

### تغییر نام کارگاه:
```javascript
WORKSHOP_DISPLAY_NAME: 'نام دلخواه'  // مثال: 'کلاس'، 'دوره'، 'کارگاه'
```

### تغییر نام‌های فیلدها:
```javascript
FIELD_DISPLAY_NAMES: {
  INSTRUCTOR_NAME: 'نام راهبر',     // نام مربی
  PHONE: 'تلفن راهبر',             // تلفن مربی
  COST: 'منطقه',                   // هزینه (منطقه)
  LINK: 'منطقه'                    // لینک (منطقه)
}
```

### فعال/غیرفعال کردن فیلدها:
```javascript
REQUIRED_FIELDS: {
  INSTRUCTOR_NAME: 1,    // نام مربی - اجباری
  PHONE: 0,              // تلفن - اختیاری  
  COST: 0,               // هزینه (منطقه) - موقتاً غیرفعال
  LINK: 1                 // لینک (منطقه) - اجباری
}
```

## ✅ مزایا

1. **مدیریت متمرکز:** همه نام‌ها در یک جا تعریف شده
2. **تغییر آسان:** بدون نیاز به تغییر کد در فایل‌های مختلف
3. **هماهنگی:** با کانفیگ نقش‌ها هماهنگ است
4. **انعطاف‌پذیری:** امکان تغییر نام‌ها برای نیازهای مختلف
5. **کنترل فیلدها:** امکان فعال/غیرفعال کردن فیلدها

## 🔧 نکات مهم

- فیلد **هزینه (منطقه)** موقتاً غیرفعال شده و در فرم‌ها نمایش داده نمی‌شود
- فیلد **لینک (منطقه)** همچنان فعال و اجباری است
- برای فعال کردن مجدد فیلد هزینه، کافی است `COST: 1` تنظیم شود
